Welcome to the Code Institute Full Stack Software Development Resource Pack! Like many of you, I am relatively new to the world of digital innovation, and through my time on Slack have come across some very useful Resources, Tools & Additional supports that have helped me in my journey to date.
My name is Anthony and as recently appointed student lead for User Centric Frontend Development, I decided to create a "webspace" dedicated to these supplementary solutions available to me, and now to you!

Resources:

Click on any of the below headers to be taken directly to the resource.

  • W3Schools

    W3 is an educational website for learning programming languages & frameworks, such as HTML, CSS & Bootstrap (and many more). All of which you will be using through UCFD!

  • MDN Mozilla

    MDN Web Docs is an evolving learning platform for Web tech & the software that powers the Web, such as HTML, CSS & Javascript to name a few.

  • Anna Greaves' Bootstrap Grid Tutorial

    Anna Greaves, a current student on the course we are all on and my predecessor as UCFD Lead, has created an incredible resource on the Bootstrap Grid. I really think Code Institute should consider putting this video in the curriculum, as I know it has helped not only me, but many MANY other students in their understanding of how to navigate the Grid!

  • Traversy Media Youtube Channel

    For the sake of this webspace, I couldn't possibly link the thousands of videos Brad from Traversy Media has made that could possibly help us in our journey, but I have linked his Crash Course on CSS3 video for you all to sink your teeth into. (P.S: There is also a HTML5 Crash course video too.)

  • Design Course Youtube Channel

    Take a dive into the mind of a Frontend, state of the art UI Designer, again I couldn't possible link all of the useful videos he has made, but I have linked Gary's video on the 2019 Frontend Developer Crash Course (HTML & CSS).

  • Markdown Cheat Sheet

    This document was a godsend for me when I was writing my README.md file for my 1st milestone and I'm sure I'll be referencing it again for all the rest of my Milestone projects. The link will take you to Adam Pritchard's Github and the repo where this cheatsheet is housed!

  • Jon Duckett's Reference Books

    Jon Duckett has created an inspirational set of reference material for anyone from the tech-savvy experienced coder to anyone with no coding experience. I bought these within my first week and have read them extensively! From HTML & CSS to Javascript & jQuery you are sure to use these throughout your entire course.

  • Bootstrap Documentation

    If you can think of an element, Bootstrap probably has it, from responsive imagery to dynamic collapsable elements. The documentation is easy to follow and you will be using Bootstrap to structure your Milestone projects to be responsive!

  • Font Awesone Iconography Documentation

    Welcome to the world of Iconography! Font Awesome has a host of free icons that you can use to uplift your html elements to a more unique and eye-popping appeal. The documentation deals with start-up, linking to the website and the icons and how to style them to fit the theme/palette of your website!

  • Elements of User Experience

    This document write up was found by a fellow student David 'H' Hubber, it is written by Jesse James Garrett and is only one chapter of an even bigger resource. It details the importance of each of the five planes of User Experience.

  • User Stories Examples

    A quick google search for User Story Examples returned thousands of search results in seconds, the link is on "How to create a good user story". For the smallest piece of work you will create during your README.md write up, it can sometimes be the hardest, give this a good run through and you should be ok to create your own!

  • How to make a Kick-Ass first Milestone!

    These documents, of which there are 3, are the go to Gold Star in terms of useful resources. Once again a special thanks to Anna Greaves, my predecessor, on creating such a benchmark of knowledge that we can all use and learn from when we feel a bit lost regarding the direction to take with our projects!
